#  Eshop Manager MVC

##  Description
Eshop Manager MVC est une application web de gestion de produits (e-commerce) développée en PHP selon l'architecture MVC (Model - View - Controller).

Elle permet :
- La gestion des produits (ajout, modification, recherche)
- L'affichage d'un catalogue
- L'inscription et la connexion des utilisateurs

##  Technologies utilisées
- PHP (Architecture MVC)
- MySQL
- HTML / CSS
- XAMPP (ou WAMP)

##  Structure du projet

- **models/** : gestion des données (Produit, Utilisateur…)
- **views/** : interfaces utilisateur (pages)
- **controllers/** : logique de traitement
- **config/** : connexion à la base de données
- **public/images/** : images des produits
- **bd_eshop.sql** : base de données
